# technicalTasks
Java, Playwright, and TestNG - UI and API test automation

# Prerequisites
- Git
- JDK 17
- Gradle

# Clone the repository
`git clone <repository-url>`

# Install dependencies
`cd <repository-directory>`
## macOS
`./gradlew build`
## Windows
`gradlew build`

# Run tests locally
`cd <repository-directory>`
## macOS
`./gradlew test`
## Windows
`gradlew test`
